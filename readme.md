1\. Database Setup



1\. Open \*\*MySQL Workbench\*\*.

2\. Create a new empty database named `expense\_tracker`.

3\. Go to \*\*Server\*\* -> \*\*Data Import\*\*.

4\. Select \*\*Import from Self-Contained File\*\*.

5\. Browse and select the file located at: `database/expense\_tracker\_db.sql` (This file is inside the 'database' folder).

6\. Click \*\*Start Import\*\*.
